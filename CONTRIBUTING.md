# Contributing

欢迎提交汉化补充、验证结果和 bug 反馈。本项目的目标不是“把所有英文机械替换掉”，而是让中文母语用户能理解 Cursor 操作界面中每个按钮、目录和常用提示的意思。

## 适合贡献的内容

- 还没汉化的可见 UI 文案。
- 翻译不自然、容易误解的文案。
- Cursor 更新后失效的替换规则。
- 运行脚本后出现的启动失败、卡死、语法错误或完整性提示问题。
- 不暴露个人隐私的界面截图。

## 不适合贡献的内容

- 全局 DOM 扫描替换。
- MutationObserver 监听全页面可见文本。
- 修改命令 ID、配置 key、枚举值、CSS class、事件名、模型 ID 或文件路径。
- 翻译用户自己的历史对话标题。
- 翻译品牌名、产品名和远程插件市场动态描述。

## 本地验证

```bash
npm run check
node ./reapply-cursor-zh-localization.js --dry-run
```

应用补丁后还需要重启 Cursor，并按照 [docs/VERIFY.md](./docs/VERIFY.md) 逐项确认。

## 提交问题时请提供

- macOS 版本。
- Cursor 版本。
- 脚本输出。
- 问题截图。
- 是否能通过回滚恢复。

请尽量裁掉邮箱、账号、历史对话标题、仓库路径等私人信息。
